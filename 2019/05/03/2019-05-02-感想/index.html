多记不如不记

用于写反思、思考的时间太多了，以至于严重干扰到了正常的学习。以后严格限制，学习的时间不准写反思，每天反思不准超出1000字。记得太多，毫无重点，毫无条理，和什么都不记并没有太大差别。

不是为了别人的看法而活，而是为了自己的责任感而活

我有我自己的责任：我的作业、团队任务、社会责任。不管多困难，也不能放弃自己的责任。

我的身体里有一个父亲，一个母亲

父亲代表严格勤奋，母亲代表理解包容。
