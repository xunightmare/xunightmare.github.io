苟延残喘

常常忍不住想，本来我在黑暗里，没心没肺的活着，你为什么要给我光明，然后又把它抽离。可是转念一想，又不能怪你。是我自己太脆弱了呀，又能怪谁呢？没办法，苟延残喘，活一天算一天吧。

放大格局，不要局限于自身，要多和不同的人接触

局限于自身，容易陷入自怨自艾中，也容易陷入纸上谈兵的境地。抓住生活中的机会，多和不同的人接触，才能够真正改变自己。面试、夏令营、团体活动、上课等等，这些都是我锻炼的机会。

放松 <- 不在意别人的看法，而在意别人的感受 <- 不执着于结果

有自己的认知，有自己的主见，不卑不亢，不骄不躁。

难过没有错

难过便任由自己难过，没必要想自己为什么难过，也没必要解决自己的难过。
