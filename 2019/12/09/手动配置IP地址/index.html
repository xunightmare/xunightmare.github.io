手动配置IP地址

Veröffentlicht am 2019-12-09 | in 计算机基础知识 | Kommentare:

为什么要手动配置IP地址？

例如：

  1. 需要使用固定IP。
  2. DHCP服务异常，无法正常访问网络。

如何手动配置IP地址？

在Windows和Linux下配置的方法可能略有不同，但原理都是一样的，需要配置以下内容：

  1. IP // 主机的唯一标识符
  2. 子网掩码 // 将IP划分为网络号和主机号两部分
  3. DNS // DNS服务器的地址，如果直接ping域名不行，但是ping对应的IP可以，则是DNS服务出了问题

Windows10

设置->网络和Internet->更改适配器选项->右键要配置的适配器->属性->Internet协议版本4（TCP/IPv4）

Linux

  1. 在文件/etc/sysconfig/network-scripts/ifcfg-ens33中配置（猜测在使用DHCP时不会使用）
  1. 图形化界面配置。图形化界面配置的原理和修改配置文件是一样的，只不过是给用户提供了底层透明的接口。

手动配置IP地址有什么坏处？

手动配置IP比较容易造成IP冲突。
