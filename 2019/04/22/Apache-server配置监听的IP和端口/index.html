Apache server配置监听的IP和端口

Veröffentlicht am 2019-04-22

背景

同一台主机上，有两个web服务，一个服务是对外开放的，而另外一个仅限内部人员使用，需要隔离开来。考虑到使用iptables限制IP访问过于死板，最后决定将内部web服务绑定到IPv6端口上，在其他人不知道IPv6地址的情况下，保证一定的安全性。

设置

Apache server可以在主要配置文件httpd.conf中设置网站绑定的IP和端口。httpd.conf文件一般在/etc/httpd/目录下。

  1. 可以只绑定IPv4

  2. 可以只绑定IPv6

  3. 可以同时绑定IPv4和IPv6（两个端口可以不同）
