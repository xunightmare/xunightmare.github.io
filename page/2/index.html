如何应对负面情绪

每次负面情绪出现的时候，不要去责怪他们，排斥他们。想象他们是你身体的一部分，当你的身体出了问题，第一想法肯定不是排斥抛弃他，而是关心他爱护他包容他，想办法慢慢解决问题。对负面情绪，我们也要关心爱护包容，慢慢引导他，这才是正确的解决方法。

不要压抑自己的个性

发现自己一直都在压抑自己的个性，不敢将真实的自己表露出来。慢慢能够理解自己的一些行为：为什么渴望能够喝醉一次；为什么想要去游乐场放声大叫；为什么在有其他人的时候不自在；为什么在床上装睡时会有莫名其妙的安全感……

以后，我要将自己的个性释放出来，说自己想要说的话，做自己想要做的事情，不再为了别人的目光而活。只要没有伤害到别人的利益，管别人怎么看呢。

服从自己的兴趣，但是也要学会自律

找到自己的兴趣所在，充满动力，沉醉其中固然是一件好事。但所谓过犹不及，要学会克制自己的欲望，自律方能自由。生活是多元的，有身体健康，有家庭，有爱情，有朋友，保持自律，平衡好这些的关系，享受生活的每分每秒，才是长远之计。

知行合一

既要从有字之书中求知识，也要从无字之书中搬真理。喜欢读书固然是好事，但是不能做只会纸上谈兵的书呆子，要把知识用到实际生活中去。

目标远大与脚踏实地

要学习萧子升，脚踏实地，不积跬步无以至千里，不要幻想一步登天，要实实在在地付出努力。
也要学习毛泽东，志向高远，敢想敢做，不要被规矩思维所束缚。

2019-04-30-感想

Veröffentlicht am 2019-04-30 | Bearbeitet am 2019-05-12 | in 随笔&日记

今天是非常有收获的一天，学到了几件非常有意义的事情。

  1. 无论你的过去发生了什么，都不能改变一点：你依然有选择的机会。我每天都会懊悔，我曾经做错了那么多的事情，让事情发展到如今这个局面。我每天也会顾影自怜，为什么上天如此不公，没有让我生长在美好的家庭，享受良好的教育，要我遭受这一切苦难。但是，现在我知道，每一刻我都在做出自我选择。我可以选择消极的自我选择，也可以选择积极的自我选择。消极的自我选择是回忆过去、强调困难，积极的自我选择只是简单地问，接下来我该干些什么。当我一次次选择积极的时候，我的生活就在慢慢改变。

    参考：【风雨哈佛路】原作者Liz Murray演讲

  2. 生活中每一件小事并不是负担，而是一种虔诚的仪式。在我抑郁的时候，我常常怀疑人生的意义。我找不到人生的意义，或者说，每次我仿佛找到了人生的意义，但是不久又会被我自己推翻。我觉得人生应该肯定是有意义的，不然为什么大家都要辛辛苦苦地活着呢！一个人每天要洗脸、刷牙、吃饭、睡觉、上厕所、洗澡洗衣服、有时候还要受到病痛的困扰，最糟糕的是，每天都有那么多的烦心事，活着真的是一件非常辛苦的事情。

    可是，当我看张春的《在另一个宇宙的1003天》的时候，我被深深地震撼了。我没有想到，我严重的生活琐事，在作者的眼中，可以如此有趣而美好。我尝试着认真虔诚地去干每一件微不足道的小事，认真叠被子，认真穿衣服，认真刷牙，认真洗脸…我带着感恩的心态看待每一件事物，我感恩被子给我温暖、感恩杯子为我刷牙、感恩牙齿为我增添光彩一直陪伴着我…我和生活中的一切事物亲切交谈。这样的事情我很少做，我做起来很不自然，常常走神，有时会感到不安，但是有时候真的会有一种很久很久没有感受到过的宁静环绕着我，我感到满足、幸福。

    参考：在另一个宇宙的1003天

  3. 一个人不能总是局限于自己身上的小格局，要将目光放到更大的格局上。我的内心有一种热情在燃烧，我听到了自己内心中一直压抑的声音：“我要为人类、为世界做一些事情”。与之相比，个人的利益、感受要微小的多。

    我一直向往爱情的美好，渴望一场轰轰烈烈的爱情，但我似乎一直不理解什么是真正的爱情。老师说，爱情不是生活的全部，我觉得她说的有道理，但是我又觉得很不舒服，像是打破了我一直以来的信仰，所谓“问世间情为何物，直教人生死相许”，这样的浪漫与美好只存在于人们的想象之中吗。现在我觉得，爱一个人是小爱，爱人类、爱世界是大爱。爱情确实是很美好很珍贵的东西，但是我想为这个世界做一些事情，为更多人带去爱。爱是一种责任，爱自己，爱别人，爱这个世界，少去想自己需要什么，多去想自己能够做些什么，付出比索取更幸福。

    参考：恰同学少年

缺爱的人如何治愈自己

Veröffentlicht am 2019-04-29 | Bearbeitet am 2019-05-12 | in 随笔&日记

缺爱对个人成年后的影响：

多层次、多方面。意识层面往往认识不到，往往是一种根源性的心理模型影响。需要重新整合性、超越性的理解和顿悟才能慢慢治愈。

  1. 个人自尊感的降低。将自己置于一种容易受到伤害的地位，这又会加固自己自尊低的认识。
  2. 童年关系外显型模式。现在人际关系往往是以幼年的人际关系为蓝本的，我们在人际关系相处中往往倾向于和与父母相像的人进行联接，我们的态度也会促使对方用父母的方式对我们。
  3. 童年关系内显性模式。童年的人际关系也会成为我们和自己相处的蓝本。例如：常常不能放松自己，总是苛责、指责自己。 或 随心所欲。以一种极端防御的方式，在头脑中表示对父母苛责、指责的反抗。
  4. 消极性的价值观模式形成。对一件事情消极或者积极的看法。消极信号和糟糕至极结果的反馈模式。

成年后该如何解决

  1. 个人要有一个意识，接纳和接受自己的过去，不管是好的还是坏的。只有接纳过去才能有能力面对未来。
  2. 自我整合（专业的指导和帮助）。
    1. 心理咨询。
    2. 自我反省、自我觉察。
  3. 意识到这些创伤对我们的影响，是我们自己选择的，是我们自己选择被影响，不管是意识还是潜意识。

O-checker: Detection of Malicious Documents through Deviation from File Format Specifications

概述：

背景：
在目标邮件攻击中，攻击者常常会诱导收信人打开恶意邮件附件，从而造成机密信息泄露等一系列危害。恶意邮件附件中，大部分都是文档文件。和可执行文件不同的是，不能通过检查文件扩展名来判断风险。
主要内容：
o-checker：一种恶意文档文件检测工具。
主要思想：
检查文档文件和文件规范的偏差。
优点：

  1. 速度快。
  2. 稳定性好。在文档中隐藏恶意代码的技术种类相对于恶意代码的种类要少得多，并且文档文件规范比文档处理器也要稳定得多。
  3. 准确率高。在实验中，o-checker获得了96.1%的准确率。

恶意文档文件的结构和一般执行过程：

  • Exploit code：利用文档处理器漏洞的程序。
  • Shell code：创建可执行文件和诱饵文档文件，并启动可执行文件。
  • Executable file
  • Decoy document file

文档结构

RTF

由控制字、控制符和组组成。

  • {} 定义一个组，组可以嵌套。
  • \ 标识一个RTF控制码
  • {\rtf ...} 一个有效的RTF文件都包含这样一个组，{是RTF代码的第一个字符，}表示文件结束符。

CFB

一个包含存储和流的嵌套层次结构的单个文件。与文件结构类比，存储对应目录，流对应文件。
物理结构：一个512字节的头部和一些等长的扇区。用一个32位非负整数标识扇区，成为扇区号。一个流被划分存储到多个扇区中。
头部中定义了FAT和DE的位置。FAT中定义了扇区链。DE中定义管理名字，大小、流，存储的父子关系。
