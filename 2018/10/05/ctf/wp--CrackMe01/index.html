writeup–CrackMe01

题目地址：https://www.ichunqiu.com/battalion?t=1&r=0

  1. 拿到这道题目的时候其实我很蒙，看起来只给了一个exe文件，应该是一道很平常的逆向题，但是用IDA打开之后才发现难度，以前常用的定位关键代码的方法全部失效了：
    • 函数窗口中找不到main函数，也找不到一个有明确意义的函数，只有一大堆不知道是什么函数的函数
    • 搜索字符串也找不到有意义的字符串

  2. 这个时候最重要的就是要定位到关键代码，运行程序的时候会有一个图形界面，图形界面中有一个输入框，我尝试随便输入一些东西，并没有反应，但是可以猜测，如果输入的是正确的flag，那么一定会有不一定的反应，很大可能就是会有一个弹框，于是尝试去找弹框的函数，先尝试了MessageBox函数没有找到关键代码，后来尝试了PostMessageW函数，向下翻找到了关键的代码：

    按F5反编译：

    发现如果v6等于0x4b6的时候，就会弹窗
    往前找可以发现v6用来和chText中的字符进行了异或，所以猜测异或得到的结果就是flag，于是写脚本验证：

    #include <stdio.h>
    #include<string.h>

int main()
{
    char flag[] = { 0xF0,0xDA,0xD7,0xD1,0x8C,0xFF,0xF5,0xFE,0xE3,0xF8,0xE7,0xFF,0xE3,0xE9,0xF0,0xF3,0x85,0x80,0x84,0xF2,0xF4,0xF3,0 };


    for (int i = 0; i < strlen(flag); i++)
    {
        printf("%c",flag[i] ^ 0x4b6);
    }


    return 0;
}
